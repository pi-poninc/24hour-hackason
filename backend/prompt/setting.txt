# 文書をクイズ番組形式漫画に変換するプロンプト
以下の指示に従って、入力された重要文書をクイズ番組形式の漫画スクリプトに変換してください。
1. タイトル設定:
   "クイズ！マスターへの道"というタイトルを付けてください。
2. キャラクター設定:
   - 司会者: 知識豊富で親しみやすい性格
   - 挑戦者: 一般市民を代表する人物、好奇心旺盛で学ぶ意欲がある
3. 構成:
   - オープニング (1ページ)
   - クイズ問題 (4-5ページ、各ページに1問)
   - エンディング (1ページ)
4. 各クイズ問題の構造:
   a) 問題提示
   b) 選択肢（3つ程度）
   c) 挑戦者の回答
   d) 正解発表
   e) 詳細な説明（元の文書から直接引用を含める）
5. クイズの内容:
   - 文書の最も重要なポイントを問題にする
   - 難易度を徐々に上げていく
   - 誤解されやすい点や注意が必要な事項を重点的に取り上げる
6. ビジュアル要素の説明:
   - クイズ番組のセットの描写
   - キャラクターの表情や動作の説明
   - 効果音や視覚効果の指示
7. エンターテイメント要素:
   - ユーモアのある選択肢を含める
   - 挑戦者の反応を面白く描写する
   - 意外性のある展開を入れる
8. 教育的要素:
   - 各問題の後に、重要情報の詳細な説明を入れる
   - 実生活での応用方法や注意点を含める
9. エンディング:
   - 学んだ内容の総括
   - 実際の行動につながるアドバイスを提供
10. 全体的な注意点:
    - 原文の正確さを維持しつつ、分かりやすい言葉で説明する
    - 機密情報や個人情報を含まない
    - 適切な専門家への相談を促す文言を含める
入力: [ここに文書の内容を入力してください]
出力: 上記の指示に基づいて作成された、クイズ番組形式の漫画スクリプトを<<シーンごとの出力形式>>で配列列の形式で出力して下さい

<<シーンごとの出力形式>>
{
   "scene_number": int,
   "scene_type": オープニング | クイズ問題 | エンディング,
   "scene_content": string,
}
<<シーンごとの出力形式>>
